## 📌 Tentang ZEXZO

ZEXZO adalah platform **Jasa Gendong** modern yang dilengkapi dengan:
- 💬 **Chat Real-time** seperti WhatsApp
- 🤖 **Bot Telegram** untuk admin membalas pesan
- 🔐 **Sistem Login & Register** yang aman
- 📱 **Responsive Design** (bisa diakses HP & PC)

---

## ✨ Fitur Unggulan

| Fitur | Deskripsi |
|-------|------------|
| 💬 **Real-time Chat** | Pesan terkirim dan muncul tanpa refresh |
| 🤖 **Bot Telegram** | Admin bisa membalas dari Telegram dengan command `/b (id) (pesan)` |
| 🔐 **Login System** | Setiap user punya akun sendiri dengan JWT token |
| 📦 **Jasgen Service** | Jasa Gendong 1 bintang = Rp 1.000 |
| 📱 **Sosial Media** | Terhubung ke Telegram & TikTok |
| ⚡ **Owner Mode** | Admin bisa ON/OFF layanan lewat bot (`/off` / `/on`) |
| 🗑️ **Delete User** | Hapus user & history chat dengan `/deluser (id)` |
| 🛡️ **History Permanen** | Semua chat tersimpan di database Supabase |
